# Seed Opportunities - Manual Guide

Since we're using Firebase, you can add these 4 opportunities directly through the Admin Panel.

## How to Seed:

1. **Login to Admin Panel**: Go to `/admin/login` and login with your admin account
2. **Navigate to Opportunities**: Click on "Opportunities" in the sidebar
3. **Click "Add Opportunity"** button
4. **Fill in the form** with the data below for each opportunity
