# 🚀 Quick Production Data Setup

## Fastest Method (2 minutes)

1. **Go to your admin dashboard**: https://apple-fam-tv.web.app/admin/dashboard

2. **Open browser console** (F12 → Console tab)

3. **Copy and paste** the entire content from `quick-seed-console.js`

4. **Press Enter** and wait for completion

5. **Refresh the page** to see your new content!

## Alternative: Manual Admin Creation

If the console method doesn't work:

1. Go to **Admin Dashboard** → **Articles**
2. Click **"Create New Article"**
3. Copy content from the sample data
4. Repeat for Events and Opportunities

## Need Your Dev Data Instead?

If you want to migrate your actual development data:

1. Use the scripts in the `/scripts` folder
2. Follow the detailed migration guide in `/scripts/README.md`
3. Configure your development Firebase credentials
4. Run the export/import process
